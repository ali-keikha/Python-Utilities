# 🛠️ Python Utilities

A collection of small command-line utilities written in Python.

I created this project to improve my Python skills by building practical tools instead of solving isolated exercises. Each utility is implemented as a separate module, making the project easier to maintain and expand over time.

## Features

- 🧮 Calculator
- 🔐 Password Generator
- 📏 Unit Converter
  - Weight
  - Distance
  - Temperature
- ⚖️ BMI Calculator
- 📂 File Organizer
- 🔤 Word And Letter Counter
- 🔎 File Duplicate
- ⏳ Age Calculator

## Getting Started

Clone the repository:

```
bash
git clone https://github.com/ali-keikha/python-utilities.git
```

Navigate to the project directory:

```
bash
cd python-utilities
```

Run the application:

```
bash
python main.py
```

This project will be updated as I continue learning Python.
